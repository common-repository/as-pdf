== Description ==

This plug-in generates PDF documents from posts. Very useful if you plan to share your posts in PDF format. Link to a PDF document is automatically generated underneath each post.

Configuration options for the plug-in can be found in WP Admin in the "Settings" section.
Setting options are as follows:

*   You can change link text
*   Font size in PDF document 
*   You can decide if you want PDF to open in browser window or if the PDF download will be forced.

Plug-in uses TCPDF to generate PDF documents. http://www.tecnick.com/public/code/cp_dpage.php?aiocp_dp=tcpdf

== Installation ==

* Download plug-in
* Unpack archive into "wp-content/plugins" directory
* Activate plugin in WP Admin
* Configure plugin in the "Settings" section of the WP Admin
